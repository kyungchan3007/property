# pinhouse-chat

Shared chatbot core logic package.

## Install

```bash
npm i pinhouse-chat
```

## Exports

- `PROMPT_RULES`
- `detectPromptType`
- `resolvePromptRule`
- `toChatRequestMessages`
- `createUserMessage`
- `createAssistantMessage`
- `createAssistantErrorMessage`
- `ChatRole`
- `ChatRequestMessage`
- `ChatHistoryMessage`
